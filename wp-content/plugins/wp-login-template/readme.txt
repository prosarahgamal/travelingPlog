== Description ==

Wordpress  Login Template is a changing custom template in the wordpress login page.This plugin will enable custom background color and custom background image in the wordpress login page.This is enable to change the custom logo in the wordpress login page.
		
There are unlimited color option to change the color and background-color  of the wordprss login area, form,login submit button,login bodoy etc.In this documentations will will cover the enough area of the option pannel for you to controll the plugin efficiently.Lets began: 


Features:

* Easly installable
* Custom background color and images
* Supper customizable 
* easyly logo login change
* Unlimited color option
* Fully responsive
* Supported all browser
		

== Installation ==

1.Extract the zip file and just drop the contents in the wp-content/plugins/ directory.
2.Activate the Plugin from Plugins page.
